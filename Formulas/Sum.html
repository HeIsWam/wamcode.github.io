SUM Formula

Sum is used in most of the cases while using Excel either shortcut or formula is used. We Use it in multiple ways. Likewise, There are multiple shortcuts and different types to do Sum. One of the Basic and common usage of Sum is by selecting the range of the cells.

Sum Range

Above image illustrates about how to make a sum from A1 cell to A8 cell. If you want to Sum multiple columns also we can do it, By selecting the range. It is Row wise Sum we can also do it in column wise.

=SUM(A1:C10) Sum from A1 to C10 =SUM(C1:C10) Column wise Sum from C1 to C10

Sum Entire Column

When we do not have specific range to select in a column then we can select entire column so that all the Values in the Column will gets added up and Text will be ignored.

=SUM(A:C) Sum multiple columns

We can also Select multiple columns just like we are doing it for Single column.

Sum Entire Row

We can also sum the entire row just like we did it for column.

=SUM(1:1) Sum complete row

Sum Multiple Rows

We can also sum the entire row just like we did it for column.

=SUM(1:12) Sum Multiple rows

Sum Different Cells

If you wanna Sum different cells then you can make use of this type.

=SUM(A3,A5,C1)Sum Values in Different Cells

You can sum any cells in any direction apart from Same cells which gives you error.
